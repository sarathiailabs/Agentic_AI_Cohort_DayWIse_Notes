## 🚀 Setup & Execution Guide

We recommend using **`uv`**, the blazing-fast Python package installer and resolver.

### Prerequisites
1. Install `uv` if you haven't already:
   ```powershell
   # Windows PowerShell:
   powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
   ```
2. Set up your environment:
   ```bash
   # Clone or enter the workspace directory
   cd sarathiAILabsTeachingNotes/Agentic_AI_Cohort_DayWIse_Notes
   
   # Synchronize virtualenv with uv
   uv venv
   .venv\Scripts\activate
   ```

### Running the Live Demos
You can run the live terminal examples directly from the command line:
```bash
# Install dependencies using uv
uv pip install -r Day5_Generative_AI_and_LLM_APIs/15_Resources.md (Wait, or use requirements.txt)
```

## ⚖️ Quality Standards & Best Practices

All content generated in this repo adheres to the following principles:
- **Beginner-Friendly, Industry-Focused**: Always starts with a tangible real-world problem and a story, before diving into complex math or code.
- **Production-Style Coding**: Avoids lazy toy examples. Code includes error handling, state memory, multi-provider interfaces, token usage logging, and rate limiting.
- **Modern Developer Aesthetics**: Formatted with high-contrast, harmonious visual tables, diagrams, tip boxes, and warnings.
